# 🏨 Hotel Management System

A full-stack Hotel Management web application with a customer-facing booking site and a dedicated admin panel for hotel staff.

**🌐 Live Site:** https://hotel-management-liard-five.vercel.app/

---

## 📌 Project Overview

This project is a Hotel Management System built for two types of users:

- **Guests** — Browse rooms, make bookings, and submit reviews
- **Admins** — Manage rooms, bookings, and reviews from a dedicated admin panel

---

## 🚀 Features

- Browse available hotel rooms
- Room booking system
- User registration and login
- Reviews — only logged-in users can submit reviews
- Admin panel to manage rooms and bookings
- Responsive design

---

## 🛠️ Tech Stack

- **Frontend:** React.js — deployed on Vercel
- **Backend:** Flask (Python) — deployed on Render
- **Database:** PostgreSQL

---

## ⚙️ Getting Started

### 1. Clone the Repository

```
git clone https://github.com/sanjay-042006/hotel_management.git
cd hotel_management
```

### 2. Run the Backend

```
cd backend
pip install -r requirements.txt
flask run
```

### 3. Run the Frontend

```
cd frontend
npm install
npm run dev
```

---

## 🖥️ Deployment

- **Frontend** is deployed on **Vercel**
- **Backend** is deployed on **Render**
